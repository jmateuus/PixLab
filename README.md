# 🚀 PixLab - Simulador de Transações Pix

O PixLab é um projeto backend desenvolvido para simular transações Pix utilizando arquitetura orientada a eventos.

O sistema é baseado em mensageria (Apache Kafka), onde transações são produzidas, consumidas e processadas de forma assíncrona, permitindo análise e visualização em tempo real.

## 🎯 Objetivos do Projeto

- Simular envios e recebimentos de Pix
- Aplicar arquitetura orientada a eventos
- Trabalhar com processamento assíncrono
- Explorar boas práticas de backend
- Servir como projeto de portfólio para o setor financeiro

## 🧱 Stack Tecnológica

- Java
- Spring Boot (ou Quarkus, se for o caso)
- Apache Kafka
- Docker
- PostgreSQL (se estiver usando)
- Maven

## 📌 Conceitos Aplicados

- Event Driven Architecture
- Producer/Consumer
- Clean Code
- Arquitetura Hexagonal (em evolução)
- Processamento assíncrono
